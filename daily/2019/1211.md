#### JMM
```
JMM(java memory model)是jvm的规范
用于屏蔽掉各种硬件和操作系统之间的访问差一，以达到java程序在各个平台都能达到一致的并发效果
规范了JVM与物理内存如何协同工作，规定了一个线程何时可以看到其它线程修改后的共享变量的值，以
及在必须时如何访问共享变量。
```
